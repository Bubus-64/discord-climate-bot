# 🌍 Discord Climate Bot

Bot Discordowy pomagający edukować o zmianach klimatu poprzez:

* 🌱 Tipy ekologiczne
* 🧠 Quizy (tak/nie)
* 🔥 Wyzwania dnia
* 😂 Losowe memy

---

## ⚙️ Komendy

| Komenda     | Opis                     |
| ----------- | ------------------------ |
| `$tip`      | Losowy tip klimatyczny   |
| `$quiz`     | Quiz (odpowiedz tak/nie) |
| `$wyzwanie` | Wyzwanie dnia            |
| `$mem`      | Losowy mem               |

---

## 🚀 Instalacja

1. Sklonuj repo:

```
git clone https://github.com/TWOJ_NICK/discord-climate-bot.git
```

2. Zainstaluj wymagania:

```
pip install -r requirements.txt
```

3. Wklej swój token:

```python
bot.run("TWÓJ_TOKEN")
```

4. Uruchom:

```
python bot.py
```
