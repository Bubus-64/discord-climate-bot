# 🌍 Discord Climate Bot

Bot Discordowy pomagający edukować o zmianach klimatu poprzez:

* 🌱 Tipy ekologiczne
* 🧠 Quizy (tak/nie)
* 🔥 Wyzwania dnia
* 😂 Losowe memy

---

## ⚙️ Komendy

| Komenda     | Opis                     |
| ----------- | ------------------------ |
| `$tip`      | Losowy tip klimatyczny   |
| `$quiz`     | Quiz (odpowiedz tak/nie) |
| `$wyzwanie` | Wyzwanie dnia            |
| `$mem`      | Losowy mem               |

---

## 🚀 Instalacja

1. Sklonuj repozytorium:

```
git clone https://github.com/Bubus-64/discord-climate-bot.git
```

2. Zainstaluj wymagania:

```
wszystkie biblioteki w kodzie
```

3. Wklej swój token:

```python
bot.run("twój token")
```

4. Uruchom:

```
python bot.py
```
